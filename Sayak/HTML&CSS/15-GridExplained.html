Grid Explained

Basic Grid Layout

The eight (8) types of grid container properties are:

  • grid-template-columns
  • grid-template-rows
  • grid-auto-columns
  • grid-auto-rows
  • justify-content
  • justify-items
  • align-content
  • align-items
